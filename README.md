## Getting Started

```sh
$ git clone git@github.com:techcampman/techdesign_kyoto_c.git
$ cd techdesign_kyoto_c
$ bundle install
$ bundle exec pod install
$ open -a Xcode TachDesiginKyotoInaka.xcworkspace
```
